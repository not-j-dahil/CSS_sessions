Length and Width!

This div will resize when window gets smaller! Try to resize, this paragraph will resize as well when the width of the window is shorter than the paragraph!

min-width: 800px
This div will only resize to the minimum width. Meaning, a horizontal scoll will appear if the div is wider than the window. Pretty cool ey!
